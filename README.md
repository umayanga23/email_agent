# Gmail AI Agent 🤖📧

> Zero-cost email automation using n8n, Groq LLM, and Gmail API

[![n8n](https://img.shields.io/badge/n8n-self--hosted-orange)](https://n8n.io)
[![Groq](https://img.shields.io/badge/Groq-llama--3.1--8b-blue)](https://console.groq.com)
[![Cost](https://img.shields.io/badge/cost-$0%2Fmonth-brightgreen)](.)
[![License](https://img.shields.io/badge/license-MIT-green)](.)

An intelligent email processing system that automatically classifies incoming Gmail messages using a free LLM, routes them to category-specific actions, and delivers a single daily digest — eliminating inbox noise entirely.

---

## What It Does

Every email that arrives in your Gmail inbox is:

1. **Classified by AI** — Groq LLM reads the email and categorises it as `JOB_VACANCY`, `URGENT`, `NEWSLETTER`, `SPAM`, `INQUIRY`, or `OTHER`
2. **Logged silently** — category, summary, and draft reply saved to Google Sheets — no email sent to you
3. **Acted upon automatically** — labelled, draft-replied, or deleted based on category
4. **Summarised once daily** — a single digest email at 9 PM with every email from the day grouped by category

---

## Architecture

### Workflow 1 — Email Agent (runs every 50 minutes)

```
Gmail Trigger
      ↓
AI Agent (Groq llama-3.1-8b-instant)
      ↓
Google Sheets — Append Row (silent log)
      ↓
Switch Node
 ├── NEWSLETTER  →  Add Gmail label
 ├── URGENT      →  Create draft reply
 ├── JOB_VACANCY →  Create draft reply + CV reference
 └── SPAM        →  Delete message
```

### Workflow 2 — Daily Digest (runs once at 9 PM)

```
Schedule Trigger (9 PM daily)
      ↓
Google Sheets — Read all rows
      ↓
Code Node — Build digest email
      ↓
Gmail — Send one digest email
      ↓
Google Sheets — Clear rows (reset for next day)
```

## AI Classification

The AI Agent returns structured JSON for every email:

```json
{
  "category": "JOB_VACANCY",
  "summary": "Software Engineer internship at Company X",
  "needs_reply": true,
  "draft_reply": "Thank you for reaching out. I am interested in this opportunity..."
}
```

Categories and actions:

| Category | Action |
|---|---|
| `JOB_VACANCY` | Create draft reply with CV reference |
| `URGENT` | Create draft reply using AI-generated text |
| `NEWSLETTER` | Add Gmail label (Promotions) |
| `SPAM` | Delete message |
| `INQUIRY` | Log only (needs_reply flagged in digest) |
| `OTHER` | Log only |

## Setup

### Prerequisites

- Node.js v18+
- A Google account
- A free [Groq account](https://console.groq.com)

### 1. Install n8n

```bash
npm install n8n -g
n8n start
```

Open `http://localhost:5678` and create a local account.

### 2. Get a free Groq API key

Go to [console.groq.com](https://console.groq.com) → API Keys → Create new key.

### 3. Enable Google APIs

In [Google Cloud Console](https://console.cloud.google.com):

- Create a new project (e.g. `n8n-gmail-agent`)
- Enable: **Gmail API**, **Google Sheets API**, **Google Drive API**
- Create OAuth 2.0 credentials → Web application
- Add redirect URI: `http://localhost:5678/rest/oauth2-credential/callback`

### 4. Import workflows into n8n

In n8n:
- Go to **Workflows** → **Import from file**
- Import `Email_Agent.json`
- Import `Daily_Email_Digest.json`

### 5. Configure credentials

In n8n → **Settings** → **Credentials**, add:

- **Gmail OAuth2** — paste Client ID + Secret → Sign in with Google
- **Google Sheets OAuth2** — same credentials
- **Groq API** — paste API key

### 6. Create Google Sheet

Create a new Google Sheet named `agent_ai` with these column headers in row 1:

```
date | From | Subject | category | summary | needs_reply | draft_reply
```

Copy the Sheet ID from the URL and update the `documentId` in both workflow nodes.

### 7. Publish both workflows

Open each workflow → click **Publish** → both run automatically on schedule.

## Known Limitations

- Groq free tier can cause intermittent timeouts under very high email volume — model changed to `llama-3.1-8b-instant` for stability
- Draft replies are saved for manual review, not auto-sent, to avoid incorrect sends
- CV attachment for job vacancy emails requires Google Drive file ID to be set manually
- Only processes emails with a readable `snippet` field from Gmail API
